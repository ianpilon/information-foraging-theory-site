# Information Scent Absorption Rate
**Summary**: Information Scent Absorption Rate (ISAR) is a method for measuring the navigability of a site by computing the probability of users reaching the desired destinations on the site.
**Sources**: The Bloodhound Project- Automating Discovery of Web Usability Issues using the InfoScent Simulator.pdf
**Last updated**: 2026-05-27
---
## Introduction
Information Scent Absorption Rate (ISAR) is a method for measuring the navigability of a site. ISAR computes the probability of users reaching the desired destinations on the site.
## Methodology
ISAR uses a combination of information retrieval techniques and spreading activation to predict the user flow throughout a site.
## Results
ISAR has been tested on four Web sites with 244 subjects over 1385 user sessions. The results show that ISAR correlates with real users surfing for information on the sites.
## Conclusion
ISAR is a useful method for measuring the navigability of a site. It can be used to improve user experience by identifying areas of a site that are difficult to navigate.
