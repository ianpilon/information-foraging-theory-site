# Web User Flow by Information Scent
**Summary**: Web User Flow by Information Scent (WUFIS) is a predictive simulation technique based on a combination of information retrieval techniques and spreading activation.
**Sources**: The Bloodhound Project- Automating Discovery of Web Usability Issues using the InfoScent™ Simulator.pdf
**Last updated**: 2026-05-27
---
## Introduction
WUFIS is a predictive simulation technique used by the Bloodhound Project to simulate user surfing behavior.
## Simulation Algorithm
The simulation algorithm uses a combination of information retrieval techniques and spreading activation to predict user flow.
