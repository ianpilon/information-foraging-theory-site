# The Bloodhound Project
**Summary**: The Bloodhound Project is a service that automatically analyzes the information cues on a Web site to produce a usability report. It uses the InfoScent™ Simulator to simulate user surfing behavior and predict the usability of a site.
**Sources**: The Bloodhound Project- Automating Discovery of Web Usability Issues using the InfoScent™ Simulator.pdf
**Last updated**: 2026-05-27

## Introduction
The Bloodhound Project is a push-button navigation analysis system that automatically analyzes the information cues on a Web site to produce a usability report. It uses the InfoScent™ Simulator to simulate user surfing behavior and predict the usability of a site.
## Information Scent Absorption Rate
The project introduces a new method called Information Scent Absorption Rate (ISAR), which measures the navigability of a site by computing the probability of users reaching the desired destinations on the site.
## Related Work
The project is related to other research on information foraging theory, such as Berrypicking and ASK.
## Conclusion
The Bloodhound Project is a unique approach to automating usability testing, using a simulation of user surfing behavior to predict the usability of a site.
