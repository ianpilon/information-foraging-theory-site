# The Bloodhound Project: Automating Web Usability Analysis
**Summary**: The Bloodhound Project is a push-button navigation analysis system that uses the InfoScent simulator to analyze the information cues on a Web site and produce a usability report. The system aims to reduce the need for human labor during usability testing and improve user experience.

## Introduction
The Bloodhound Project is a prototype service that uses the InfoScent simulator to analyze the information cues on a Web site and produce a usability report. The system is designed to reduce the need for human labor during usability testing and improve user experience.
## Methodology
The Bloodhound Project uses the InfoScent simulator, which is based on the concept of information scent. Information scent is the user's perception of the value and cost of accessing a particular piece of information. The simulator uses a combination of information retrieval techniques and spreading activation to predict the user flow throughout a site.
## Results
The Bloodhound Project has been tested on four Web sites with 244 subjects over 1385 user sessions. The results show that the system correlates with real users surfing for information on the sites.
## Conclusion
The Bloodhound Project is a unique approach to automating web usability analysis. The system uses the InfoScent simulator to analyze the information cues on a Web site and produce a usability report. The results show that the system is effective in predicting user behavior and can be used to improve user experience.
